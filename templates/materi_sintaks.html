1. Variabel & Penulisan Sintaks Dasar

Python menggunakan tanda = untuk menyimpan nilai ke dalam variabel.

nama = "Andi"
umur = 20

2. Tipe Data Dasar

  • int: Bilangan bulat, contoh: 10
  • float: Bilangan desimal, contoh: 3.14
  • str: Teks/string, contoh: "Halo"
  • bool: True/False, contoh: True
  • list: Kumpulan data, contoh: [1, 2, 3]
  • tuple: Kumpulan data tetap, contoh: (1, 2, 3)
  • dict: Data pasangan kunci-nilai, contoh: {"nama": "Andi", "umur": 20}

Contoh Penggunaan

nama = "Budi"
umur = 25
nilai = 87.5
lulus = True
daftar_nilai = [80, 85, 90]
data_mahasiswa = {"nama": "Budi", "umur": 25}
